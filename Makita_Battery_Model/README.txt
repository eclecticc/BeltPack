# Summary

Makita Battery Model.

Everbilt 3-1/2 in. x 15 in. Stainless Steel Push Plate is readily available and only $10.

is about 1mm thick, just right for the terminal connector blades.

but I'm not sure how conductive stainless steel is, or whether it will be easy to solder.
